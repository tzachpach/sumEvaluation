
This is a dataset accompanying the publication:

   Ranking Generated Summaries by Correctness: An Interesting but Challenging Application for Natural Language Inference
   Tobias Falke, Leonardo F. R. Ribeiro, Prasetya Ajie Utama, Ido Dagan and Iryna Gurevych
   ACL 2019

This document describes version 1.0 of the data.
   
The archive contains the following files:

- test_*_org: crowdsourced correctness evaluation presented in Section 4 / Table 1
- val_reranking: crowdsourced annotations for summary reranking (used for Table 2 upper part)
- test_*_reranked: crowdsourced correctness evaluation after reranking on test (Table 2 lower part)
- val_sentence_pairs: derived triples for sentence ranking (used for Table 3)

The data is released under the Creative Commons CC-BY-SA 4.0 license.
